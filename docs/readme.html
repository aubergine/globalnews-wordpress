Plug-in Used

  1. Yet Another Related Posts Plug-in - We used YARPP to display related posts based on relevancy of article.
  2. Customizable Post Listings - to display recently commented posts.
  3. WP Email - for email post to friend feature. You can use direct mail to option also.

These are only 3 plug-in you need to install to use in Global News theme. Also there may be number of alternatives for these functions but we choose these three because all these plug-in work great, easy to install/setup and supported by developers.

Note: In YARPP plug-in settings turn off the option "Automatically display related posts?"

Installing Global News Theme

  1. Download latest package of Global News Theme from here.
  2. Extract files on your HDD.
  3. Upload globalnews folder on your webhost with the help of any FTP client.
  4. Login to your wordpress admin panel.
  5. Activate Global News under Appearances > Themes
  6. That's it. You are on!

Setting Up Global News Theme

You might want to setup your theme before you make it online. It's too easy because you can setup theme settings from wordpress admin panel. Go to Appearances > GlobalNews Options

There will be 2 main group settings. Each group contains number of options for easy setup.

  • News Settings - configure your news articles from here.
  • Advertisement Blocks Settings - configure advertisement blocks.

A. News Settings

  • Weather Location - You can enter location zip code of your place to show latest weather updates of your place. You can show weather forecast of your location in header weather widget. Don't know your zip code? Visit here.

  • Newsticker Articles - This option lets you configure number of articles you want to show in header news ticker at home page. Default value is 20.

  • Featured Articles - This option lets you configure number of articles you want to show as featured location on home page. Default value is 1.

  • Headline Articles - This option lets you configure number of articles you want to show in headline section right to featured article section. Default value is 3.

  • News Carousel Articles - This option lets you configure number of articles you want to show in article carousel just below the featured article section on home page. Default value is 6.

  • Newsflash Articles - This option lets you configure number of articles you want to show in each newsflash box located just below article carousel on home page. Default value is 2.

B. Advertisement Blocks Settings

These settings are primarily for various advertisement blocks on this theme. We have marked advertisement blocks in this image.


  • Frontpage 120x600 Advertisement - Code for ad blocks A (120x600) on homepage at top right.
  • Frontpage 300x250 Primary Advertisement - Code for ad blocks B (300x250) on homepage at right footer.
  • Subpage 300x250 Primary Advertisement - Code for ad blocks C (300x250) on post page at top right sidebar.
  • Subpage 300x250 Secondary Advertisement - Code for ad blocks D (300x250) on post page at middle right sidebar.
  • Subpage 300x250 footer Advertisement - Code for ad blocks E (300x250) on post page at right footer.

You can your own custom html code for these ad blocks locations or alternatively you can also use Google Adsense code.

How to Assign/Post News in Different Sections

News in all news blocks are controlled by tags. You just need to add tags with each post you want to be displayed in different sections. For example if you want to show a article at featured location then you just add "featured" tag with a post and it will be displayed in featured section. If there are more articles then your limit (number of articles for that section) then most recent article will be displayed. This goes with all other section which I am going to explain here.

How to Upload Images for a Post

  1. To upload image, you can use the built-in Add Image WP feature. In the post editing page, notice the Add media icons just below your post title and permalink. Click “Add Image” and an image uploading page will show up.

  2. A new popup box will open. click select files button.

  3. Upload any image that you like.

  4. After the image is uploaded, you can see the image location from the ‘Link URL’ field. In this tutorial, it is: http://www.instantshift.com/wp-content/uploads/2009/06/1.png. Values may vary depending on your blog settings. One may have the time-structured folder while others may not. Copy that value but remember to get rid of your blog home page location. In my case, the final value will be wp-content/uploads/2009/06/1.png.

    Then, close the image uploading page and you do not need to clik ‘Insert to Post’ button if you do not want to. When you’re back in the post editing page, scroll down until you find “Custom Fields”. Make a new Custom Field with "thumb" as the key and the image location as the "Value".
